Privacy Policy

Last updated: June 2026

Hemklok is an AI buyer's agent for the Swedish housing market. This policy explains what data we collect and how we use it.

What we collect

Hemklok does not collect, store, or sell personal data. The service is a read-only AI assistant — it reads publicly available real estate market data and responds to your questions. No account is required and no information you share in the conversation is retained after the session ends.

The ChatGPT GPT

The Hemklok beta runs as a custom GPT on ChatGPT (OpenAI). Conversations are processed by OpenAI under their own Privacy Policy and Usage Policies. We do not receive or store any conversation data from OpenAI.

Real estate data API

When you use the Hemklok GPT, it may call our real estate data API to retrieve average sale prices from Booli's completed-transactions dataset. This API call contains only query parameters (year, number of rooms) — no personal information is transmitted.

This website

llm-brf.github.io is a static site hosted on GitHub Pages. We do not run analytics, tracking pixels, or advertising scripts. GitHub may collect standard server logs as described in GitHub's Privacy Statement.

Cookies

This website does not set any cookies.
